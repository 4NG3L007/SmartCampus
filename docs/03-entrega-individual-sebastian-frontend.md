## Funcionalidad desarrollada

Desarrollo del prototipo frontend de escritorio para el sistema SmartCampus.

La funcionalidad desarrollada permite mostrar:

- Panel docente.
- Listado de materias asignadas.
- Registro visual de asistencia.
- Carga visual de notas.
- Panel estudiante.
- Tabla de materias con asistencia, promedio y estado.
- Dashboard basico de direccion con indicadores simulados.

## Tecnologias usadas

- HTML5 para la estructura de las pantallas.
- CSS3 para el diseno visual.
- JavaScript para navegacion, cambio de vistas, tabs y datos simulados.
