# SmartCampus - Prototipo frontend

Este prototipo esta pensado solo para PC/escritorio.

## Tecnologias usadas

- HTML5 para la estructura.
- CSS3 para el diseno.
- JavaScript para navegacion y datos simulados.

## Estructura

```text
frontend-prototipo/
  index.html
  css/
    styles.css
  js/
    app.js
```

## Como abrirlo

Abrir `index.html` en el navegador. No necesita instalacion ni servidor.

## Que demuestra

- Panel docente.
- Materias asignadas.
- Registro visual de asistencia.
- Carga visual de notas.
- Panel estudiante.
- Dashboard basico de direccion.
